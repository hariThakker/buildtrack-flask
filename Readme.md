## Construction Project Management API

## Run Instaructions :- 

1. python3 -m venv venv
2. For mac/linux users :-
   source venv/bin/activate
   For Windows users :-
   venv/Scripts/activate
3. Install Dependencies :- pip install -r requirements.txt
4. Server will be showed running on the port showed
5. To stop the server Ctrl + C
6. to end the venv :- just type "deactivate" in terminal

## Project Details :- 

A Flask-based backend system designed to help small construction businesses
manage projects, material expenses, labour payments, and cost summaries.

### Features
- Project management
- Material expense tracking
- Labour payment tracking
- Project-wise expense summary

### Tech Stack
- Python
- Flask
- Flask-SQLAlchemy
- SQLite

### API Testing
- Thunder Client
